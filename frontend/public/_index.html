Página: Amigurumi

Ao clicar no botão "Amigurumi" no topo da página, você poderá ver todos os amigurumis disponíveis!

Para buscar uma receita, digite o nome do amigurumi na barra de pesquisa ou clique em "Novo Amigurumi" para adicionar uma nova receita

Para mais detalhes sobre uma receita, clique em "Ver Mais"




Página: Receitas

Geral

Para alterar os dados do amigurumi, clique em "Editar Amigurumi"

Para adicionar ou alterar as imagens do amigurumi, clique em "Editar Imagem"

Para adicionar uma nova receita atrelado ao amigurumi, clique em "Adicionar Receita Relacionada"

Para salvar essa receita em PDF, basta clicar em "Salvar PDF"

Para excluir todos os dados do amigurumi, clique em "Excluir Amigurumi"

Elementos

Primeiro deve-se adicionar um novo elemento do amigurumi, ao clicar no botão "Adicionar", localizado abaixo da tabela Elementos

Caso queira, alterar ou deletar algum elemento, clique nos botões "Alterar" e "Deletar", reespectivamente, que estão ao lado da linha

Tabela de Receita

Para adicionar uma nova linha em alguma tabela de receita, clique no botão "Adicionar", que se encontra abaixo de cada reespectiva tabela de receita

Nas tabelas de receitas tem a opção de alterar ou excluir alguma linha, pelos botões "Alterar" e "Deletar", reespectivamente, que estão ao lado da linha

Materiais

Para adicionar um novo material utilizado na construção do amigurumi, é só clicar no botão "Adicionar", localizado abaixo da tabela

Para alterar ou deletar algum material, clique nos botões "Alterar" e "Deletar", reespectivamente, que estão ao lado da linha

Para alterar a lista de materiais, selecione em "Escolha a Lista de Materiais", para selecionar uma lista já cadastrada

Receita Relacionada

Para visualizar receitas complementares, a receita original, veja as opções ao final da página, e clique em "Ver Mais", para ver as receitas
